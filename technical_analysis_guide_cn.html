早晨之星
阴线 + 小实体/十字星 + 阳线，三日底部反转形态。第三日阳线收盘需回补第一日阴线实体一半以上。
锤子线
低位出现，实体在上方，上影线极短或无，下影线很长（≥2倍实体）。代表空头砸盘后多头强力接盘，底部反转信号。
看跌组合形态
看跌吞没
大阴线实体完全包住前日阳线，空头强力反攻。出现在上涨趋势末端时可靠性高，量能放大更佳。
黄昏之星
阳线 + 小实体/十字星 + 阴线，三日顶部反转形态。第三日阴线收盘需回补第一日阳线实体一半以上。
流星线
高位出现，实体在下方，下影线极短或无，上影线很长（≥2倍实体）。代表多头拉高后遭空头强力打压，顶部反转信号。

🔄 筑底期 — 积蓄阶段
市场特征
  • 股价长期下跌后趋于平稳
  • 成交量极度萎缩
  • 均线由散乱趋于走平
  • RSI 在 20-40 区间徘徊
  • MACD 柱线零轴下方缩短
操作策略
  • 耐心等待，不追跌
  • 关注放量异动信号
  • 试探性小仓建仓
  • 止损设在近期低点下方
  • 等待突破确认再加仓
📈 上升期 — 趋势阶段
市场特征
  • 价格突破均线系统向上
  • 均线由平转向多头排列
  • 成交量放大配合上涨
  • MACD 金叉并站上零轴
  • RSI 保持在 50 以上
操作策略
  • 顺势做多，持股为主
  • 回调至均线附近加仓
  • 不轻易做空
  • 用均线追踪止盈
  • 出现顶部信号再减仓
⚖️ 筑顶期 — 分配阶段
市场特征
  • 价格在高位横盘震荡
  • 放量但价格无法创新高
  • 均线开始纠缠走平
  • MACD 顶背离信号出现
  • RSI 高位钝化或背离
操作策略
  • 逐步减仓，锁定利润
  • 不追涨，不抄顶
  • 严格执行止损
  • 背离信号出现时清仓
  • 等待方向明朗再入场
📉 下降期 — 下跌阶段
市场特征
  • 价格跌破均线支撑
  • 均线空头排列，死叉连续
  • 每次反弹量缩价跌
  • MACD 零轴下方运行
  • RSI 长时间 20-50 区间
操作策略
  • 空仓观望为主
  • 反弹减仓，不接刀
  • 切忌抄底操作
  • 等待筑底信号出现
  • 做空需确认趋势后
背离信号详解
底背离 — 买入信号
价格创新低，MACD / RSI 指标未同步创新低，下跌动能减弱，可能反转向上。
顶背离 — 卖出信号
价格创新高，MACD / RSI 指标未同步创新高，上涨动能衰竭，可能反转向下。

葛兰碧八大法则（均线）
买① 均线走平或上翘，价格从下方向上穿越均线 → 买入
买② 均线上升，价格暂时跌至均线附近后回升 → 加仓买入
买③ 均线上升，价格跌破均线后迅速回升 → 逢低买入
买④ 均线下跌，价格远低于均线急速下跌 → 超卖反弹买入
卖⑤ 均线走平或下弯，价格从上方向下穿越均线 → 卖出
卖⑥ 均线下降，价格暂时升至均线附近后回落 → 减仓卖出
卖⑦ 均线下降，价格升破均线后迅速回落 → 逢高卖出
卖⑧ 均线上升，价格远高于均线持续上升 → 超买考虑减仓
风险管理与常见误区
风险管理铁律
单笔止损不超过本金 2%
同向仓位总风险不超过 6%
盈亏比要求至少 2:1 再入场
连续亏损 3 次强制停止交易
盈利时使用追踪止损保护利润
分批建仓，分批离场，降低风险
常见误区
指标钝化
强势行情中 RSI、KDJ 可长期处于超买区，不能单纯依此做空，需结合趋势类指标。
假突破
价格突破必须配合成交量放大，缩量突破通常为假信号，风险极高。
信号滞后
均线和 MACD 均基于历史数据，行情快速变化时信号滞后，须结合市场背景预判。
⚠
技术指标为辅助工具，单一信号可靠性有限。建议 3-5 个指标共振确认，配合基本面分析，严格执行止损策略。
