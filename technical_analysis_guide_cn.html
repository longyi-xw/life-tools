🔄 筑底期 — 积蓄阶段
市场特征
  • 股价长期下跌后趋于平稳
  • 成交量极度萎缩
  • 均线由散乱趋于走平
  • RSI 在 20-40 区间徘徊
  • MACD 柱线在零轴下方缩短
操作策略
  • 耐心等待，不追跌
  • 关注放量异动
  • 试探性小仓建仓
  • 止损位设在近期低点下方
  • 等待突破确认再加仓
📈 上升期 — 趋势阶段
市场特征
  • 价格突破均线系统向上
  • 均线由平转向多头排列
  • 成交量放大配合上涨
  • MACD 金叉并站上零轴
  • RSI 保持在 50 以上
操作策略
  • 顺势做多，持股为主
  • 回调至均线附近加仓
  • 不轻易做空
  • 用均线追踪止盈
  • 出现顶部信号再减仓
⚖️ 筑顶期 — 分配阶段
市场特征
  • 价格在高位横盘震荡
  • 放量但价格无法创新高
  • 均线开始纠缠走平
  • MACD 顶背离信号出现
  • RSI 高位钝化或背离
操作策略
  • 逐步减仓，锁定利润
  • 不追涨，不抄顶
  • 严格执行止损
  • 背离信号出现时清仓
  • 等待方向明朗再入场
📉 下降期 — 下跌阶段
市场特征
  • 价格跌破均线支撑
  • 均线空头排列，死叉连续
  • 每次反弹量缩价跌
  • MACD 零轴下方运行
  • RSI 长时间处于 20-50 区间
操作策略
  • 空仓观望为主
  • 反弹减仓，不接刀
  • 切忌抄底操作
  • 等待筑底信号出现
  • 做空市场只在确认下降趋势后
背离信号详解
底背离 — 买入信号
价格创新低，而 MACD / RSI 指标未能同步创新低，说明下跌动能减弱，可能即将反转向上。
顶背离 — 卖出信号
价格创新高，而 MACD / RSI 指标未能同步创新高，说明上涨动能衰竭，可能即将反转向下。

核心操作原则
葛兰碧八大法则（均线）
① 均线走平或上翘，价格从下方向上穿越均线 → 买入
② 均线上升，价格暂时跌至均线附近回升 → 买入
③ 均线上升，价格跌破均线后迅速回升 → 买入
④ 均线下跌，价格远低于均线急速下跌 → 超卖买入
⑤ 均线走平或下弯，价格从上方向下穿越均线 → 卖出
⑥ 均线下降，价格暂时升至均线附近回落 → 卖出
⑦ 均线下降，价格升破均线后迅速回落 → 卖出
⑧ 均线上升，价格远高于均线持续上升 → 超买卖出
风险管理铁律
单笔止损不超过本金 2%
同向仓位总风险不超过 6%
盈亏比要求至少 2:1 再入场
连续亏损 3 次强制停止交易
盈利时使用追踪止损
分批建仓，分批离场
⚠
技术指标为辅助工具，存在滞后性。单一指标信号可靠性有限，建议 3-5 个指标共振确认，配合基本面分析，严格执行止损策略。
常见误区
指标钝化
强势行情中，RSI、KDJ 可长期处于超买区，不能单纯依此做空。
假突破
价格突破必须配合成交量放大，缩量突破通常为假信号，风险极高。
滞后性
均线和MACD基于历史数据，行情快速变化时，信号会明显滞后，须提前预判。
